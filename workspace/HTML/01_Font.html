태그의 구성 : 태그 또는 텍스트(데이터) 구성 된다.
태그의 역할 : 데이터 구분자 역할, 기능적 역할 태그는 속성을 가질수 있다.
글꼴,문단밎 목록, table : CSS 대체 가능
이미지,영상,오디오 : 상대경로
페이지 이동 : a 태그 사용

글꼴 서식

SIZE h1 > h2 > h3 > h5 > h6

h태그는 자동 줄 바꿈

제목을 표시할때 사용함


b태그는 굵게
i태그는 이탤릭체

위첨자 SUP태그
ins태그는 밑줄
